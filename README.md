```mermaid
flowchart TD
    A[多視点全方位画像] -->|全方位画像| B[多視点透視投影画像群の<br>生成]
    B -->|透視投影画像| C{骨格検出}
    C -->|有| D[注視点をグルーピング]
    C -->|無| E[終了]
    D -->|グルーピングした注視点の中心座標| F[透視投影画像の生成]
    F -->|透視投影画像| G{骨格検出}
    G -->|有| H[スケーリング]
    G -->|無| I[終了]
    H -->|スケーリングした<br>透視投影画像| J{視点間の<br>特徴点マッチング}
    J -->|マッチング有| K[バレットタイム映像]
    J -->|マッチング無| L[終了]
